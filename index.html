Github repository

English example

What is nehan.js?

Nehan.js is dynamic and logical layout engine written in javascript, generates logical paged media asyncronously.

In other words, it's a virtual browser engine over native browser engine.

Why logical layout?

Generally english or latin document flows left to right at inline level, and top to bottom at block level, but there are more format in the world, especially Asian people uses vertical document that flows top to bottom at inline level, and left to right or right to left at block level.

So we should treat document as logical format, to generate both vertical and horizontal layout from one source.

So why nehan.js?

nehan.js generates abstract logical layout tree from html string, so both horizontal and vertical layout can be generated from one same tree.

Furthermore, unlike pdf or word document, each pages are generated asyncronously, so client window is not blocked unless all pages are generated.

As soon as first page is generated, ui event is surely ready with lightning speed, and it's a huge UX advantage!

Japanese example

nehan.jsとは？

涅槃は動的で論理的なページメディアをノンブロッキングで出力できる javascriptの組版エンジンです。言い方を変えれば、ネイティブなブラウザ上にjavascript 製のブラウザを作っているようなものです。

どうして論理的なレイアウトなの？

世の中には横書きだけではなく、縦書きの文章もあります。特にアジアでは上から下にインラインテキストが流れる縦書きがよく使われます。

なのでドキュメントを横書きを前提にした通常の方式ではなく、縦にも横にも統一的に記述できる論理的なレイアウトで記述することが重要です。

そうすれば、一つのソースから縦書きも横書きも出力できるからです。

組版エンジンがjsだと何が嬉しいの？

涅槃は HTML のテキストから、縦書き横書きに依存しない論理的で抽象的なレイアウトツリーを作成します。

そのツリーは縦にも横にも出力でき、しかも表示されるまでにユーザーの画面をブロックしません。

先頭のページが表示された時点でユーザーに処理が返されるので、これは UX として非常に大きなアドバンテージです。
